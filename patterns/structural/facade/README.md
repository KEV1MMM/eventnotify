# Facade

## Nombre
Facade

## Categoría
Estructural (Structural)

## Propósito
Proveer una interfaz simple y unificada a un subsistema complejo para reducir el acoplamiento y facilitar su uso.

## Código de Referencia

### Implementación en Java
**Archivo:** `code/java/Demo.java`

**Compilar:**
```bash
cd patterns/structural/facade/code/java
javac Demo.java
```
**Ejecutar:**
```bash
java Demo
```

### Implementación en JavaScript
**Archivo:** `code/javascript/demo.js`
**Ejecutar:**
```bash
cd patterns/structural/facade/code/javascript
node demo.js
```
