# Flyweight

## Nombre
Flyweight

## Categoría
Estructural (Structural)

## Propósito
Reducir el uso de memoria compartiendo estado intrínseco entre muchos objetos similares y delegando el estado extrínseco al cliente.

## Código de Referencia

### Implementación en Java
**Archivo:** `code/java/Demo.java`

**Compilar:**
```bash
cd patterns/structural/flyweight/code/java
javac Demo.java
```
**Ejecutar:**
```bash
java Demo
```
### Implementación en JavaScript
**Archivo:** `code/javascript/demo.js`

**Ejecutar:**
```bash
cd patterns/structural/flyweight/code/javascript
node demo.js
```
