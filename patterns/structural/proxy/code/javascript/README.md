# Proxy

## 1) Nombre del patrón
Proxy

## 2) Categoría
Estructural (Structural)

## 3) Propósito (1 oración)
Proporciona un sustituto o representante de un objeto para controlar el acceso a él, pudiendo aplicar carga diferida (lazy loading), cache, permisos o logging.
