# Proxy

## 1) Nombre del patrón
Proxy

## 2) Categoría
Estructural (Structural)

## 3) Propósito (1 oración)
Controla el acceso a un objeto real mediante un sustituto que puede agregar seguridad, lazy loading o cache sin cambiar al cliente.
