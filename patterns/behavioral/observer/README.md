# Observer

**Nombre:** Observer  
**Categoría:** Comportamiento  
**Propósito:** Permite que múltiples objetos se suscriban a un evento y sean notificados automáticamente cuando ocurre, reduciendo el acoplamiento entre componentes.

## Código de Referencia (2 lenguajes)

### Java
Ubicación: `code/java/`
- Compilar: `javac *.java`
- Ejecutar: `java Demo`

### JavaScript (Node)
Ubicación: `code/javascript/`
- Ejecutar: `node demo.js`
