# Mediator

## Nombre
Mediator

## Categoría
Comportamiento (Behavioral)

## Propósito
Centralizar la comunicación entre múltiples objetos para reducir el acoplamiento directo entre ellos.

## Código de Referencia

### Implementación en Java
**Archivo:** `code/java/Demo.java`

**Compilar:**
```bash
cd patterns/behavioral/mediator/code/java
javac Demo.java
```
### Implementación en JavaScript
**Archivo:** `code/javascript/demo.js`

**Ejecutar:**
```bash
cd patterns/behavioral/mediator/code/javascript
node demo.js
```
