# Template Method

## Nombre
Template Method

## Categoría
Comportamiento (Behavioral)

## Propósito
Definir el esqueleto de un algoritmo en una clase base y permitir que las subclases redefinan pasos específicos sin cambiar la estructura general.

## Código de Referencia

### Implementación en Java
**Archivo:** `code/java/Demo.java`

**Compilar:**
```bash
cd patterns/behavioral/template-method/code/java
javac Demo.java
```
**Ejecutar:**
```bash
java Demo
```
### Implementación en JavaScript
**Archivo:** `code/javascript/demo.js`

**Ejecutar:**
```bash
cd patterns/behavioral/template-method/code/javascript
node demo.js
```
