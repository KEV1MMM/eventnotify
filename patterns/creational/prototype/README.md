# Prototype

## Nombre
Prototype

## Categoría
Creacional (Creational)

## Propósito
Crear nuevos objetos clonando una instancia prototipo para evitar construcciones costosas y simplificar la creación.

## Código de Referencia

### Implementación en Java
**Archivo:** `code/java/Demo.java`

**Compilar:**
```bash
cd patterns/creational/prototype/code/java
javac Demo.java
```
**Ejecutar:**
```bash
java Demo
```
### Implementación en JavaScript
**Archivo:** `code/javascript/demo.js`

**Ejecutar:**
```bash
cd patterns/creational/prototype/code/javascript
node demo.js
```
